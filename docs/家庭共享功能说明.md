# 家庭共享功能使用说明

## 功能概述

家庭共享功能允许用户创建房子并邀请家人一起管理物品有效期。

## 使用方法

### 1. 打开设置面板

登录后，点击顶部左侧的「⚙️」设置按钮打开设置面板。

### 2. 邀请家人

**前提条件：** 必须是房子的房主

1. 在设置面板中，点击「🎫 生成邀请码」按钮
2. 系统会生成一个6位邀请码（如：ABC123）
3. 点击复制按钮或直接分享邀请码给家人
4. 邀请码有效期为24小时

### 3. 加入房子

**前提条件：** 已登录账号

1. 打开设置面板
2. 在「🚪 加入房子」区域输入6位邀请码
3. 点击「加入」按钮
4. 成功后会自动切换到该房子，并加载对应数据

### 4. 房子管理

- **创建新房：** 点击「➕ 创建新房子」可创建新的房子
- **切换房子：** 如果属于多个房子，顶部会显示下拉框用于切换

## 数据库表结构

### invite_codes 表

```sql
CREATE TABLE invite_codes (
    id UUID PRIMARY KEY,
    code TEXT UNIQUE NOT NULL,        -- 邀请码
    house_id UUID REFERENCES houses,   -- 关联的房子
    created_by UUID,                   -- 创建者
    expires_at TIMESTAMPTZ,            -- 过期时间
    created_at TIMESTAMPTZ DEFAULT NOW()
);
```

### 迁移 SQL 文件

位于 `supabase/migrations/001_create_invite_codes.sql`

## 技术实现

- 邀请码同时支持数据库和 localStorage 两种存储方式
- 数据库表优先，失败时降级到 localStorage
- 房主可以生成邀请码，成员只能查看和加入
- 成员列表会显示房主和所有成员信息

## 注意事项

1. 每个用户可以属于多个房子
2. 物品数据按房子隔离
3. 邀请码24小时后自动失效
4. 需要在 Supabase 后台执行迁移 SQL 创建 invite_codes 表
